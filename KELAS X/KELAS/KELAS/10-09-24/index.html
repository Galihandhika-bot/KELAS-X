belajar html dan css

html digunakan untuk membuat tampilan website. css digunakan untuk mengatur tampilan.

materi html

  • h1 untuk : judul
  • p untuk : paaragraf
  • image : untuk gambar/li>
  • table : untuk membuat tabel
  • input : untuk membuat form
    • li : untuk membuat list

materi css

  1. margin,padding,position : untuk mengatur jarak antara elemen html
  2. display : untuk mengatur elemen html agar tampil/tidak tampil,mendatar/menurun
  3. style : untuk mengatur font
